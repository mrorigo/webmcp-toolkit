Progressive Integration

Three tiers. One SDK.

Start simple, scale to full autonomy. Each level builds on the last, so you adopt exactly as much as your application needs.

🥉 Bronze — Explicit Tools

Register any function as a WebMCP tool

Wrap an existing API call or function. The SDK handles the boilerplate: JSON Schema conversion, argument validation, and clean error boundaries. Any object with a parse(data) method — including Zod schemas — works as the schema.

mcp.tools.register({
  name: "add_to_cart",
  description: "Adds a product",
  schema: { parse: (d) => d, _shape: {...} },
  execute: async ({ productId }) => {
    return cart.add(productId);
  }
});
🥈 Silver — Declarative Polyfill

Semantic HTML → AI tool. Zero JavaScript.

Annotate your existing <form> elements with toolname and tooldescription attributes. The MutationObserver-powered polyfill auto-registers them with the browser's modelContext, live — even for dynamically-rendered SPA forms.

<!-- Pure HTML. No JS needed. -->
<form
  toolname="book_flight"
  tooldescription="Books a seat">
  <input
    name="destination"
    toolparamdescription="City"
    required />
</form>
🥇 Gold — Universal Delegate

Full autonomous in-page sub-agent ⚠ Experimental

Register the delegate_page_task macro-tool. An external agent sends a plain-text goal; your embedded In-Page Agent runs a full Observe → Think → Act ReAct loop over your live DOM — with Human-In-The-Loop authorization before critical actions.

mcp.enableUniversalDelegate({
  agent: new InPageAgent({
    llmProvider: provider,
    maxSteps: 15
  }),
  requireConfirmationFor: [
    "form[toolname='checkout']"
  ]
});

Built for the modern web

Everything you need.
Nothing you don't.

📦

Zero dependencies

The entire SDK ships with no runtime dependencies. No Zod, no node polyfills, no bloat. Just clean TypeScript compiled to a self-contained IIFE.

⚡

15 kb minified

The full bundle — ReAct agent, LLM providers, and declarative polyfill — weighs 15 kb. Compress with Brotli and you're under 5 kb over the wire.

🔒

Human-In-The-Loop

Define CSS selectors for elements that require user confirmation. The agent loop pauses and calls client.requestUserInteraction() before proceeding.

🧠

Two LLM providers

ChromePromptProvider for zero-cost on-device inference via globalThis.LanguageModel. OpenAIProvider for BYOK cloud fallback — but note: API keys passed to OpenAIProvider are visible in frontend JS. Prefer the local provider or proxy via a backend endpoint in production.

🔄

Live DOM observation

MutationObserver tracks newly-rendered forms and elements in real time, so tools register automatically in SPAs — no page refresh required.

📐

Future-proof standard

Built on the emerging WebMCP spec. The declarative HTML attributes are polyfilled today; when browsers adopt them natively, the library steps aside automatically.
